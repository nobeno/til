### WinRMとは

#### 概要
- 正式名称：Windows Remote Management(WinRM) 
- WinRMでは、コマンド操作が可能
- 多数のサーバで同じ操作を繰り返す作業をバッチ処理として自動化可能
- Windows Server 2012以降であれば、デフォルトで有効（デフォルトポートは5985）
- ファイヤーウォールの設定が必要かも
- macOSからも操作可能（PowerShell Coreのインストールが必要）
